# SIMRec


Implementation of SIMRec in PyTorch

This repository is a clean and readable implementation of the paper:

### ***Attribute simulation for item embedding enhancement in multi-interest recommendation***

```jsx
Liu, Y., Zhang, X., Zou, M., & Feng, Z. (2024, March). Attribute simulation for item embedding enhancement in multi-interest recommendation. In Proceedings of the 17th ACM international conference on web search and data mining (pp. 482-491).
```
